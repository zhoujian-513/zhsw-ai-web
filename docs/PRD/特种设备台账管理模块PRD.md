## 2. 产品背景

### 2.1 产品定位

特种设备台账管理模块是安全管理系统的核心子模块，用于对企业内部所有特种设备进行全生命周期管理，包括设备信息维护、定期检验管理、校验记录追踪等功能，确保特种设备的安全运行和合规管理。

### 2.2 目标用户

| 用户角色 | 职责描述 | 主要操作 |
|---------|---------|---------|
| 安全管理员 | 负责特种设备台账的日常维护和管理 | 编辑设备信息、查看校验记录、导入导出数据 |
| 设备管理员 | 负责设备信息的录入和更新 | 查看设备详情、维护校验记录 |
| 系统管理员 | 负责系统的整体配置和数据管理 | 批量导入设备数据、导出报表 |
| 审计人员 | 负责设备管理的合规性审查 | 查看设备详情、导出数据 |

### 2.3 产品目标

1. **合规管理**：确保特种设备按照国家法规要求进行登记和定期检验
2. **风险预警**：通过校验状态监控，提前发现即将到期或已过期的设备
3. **信息集中**：建立统一的特种设备信息库，便于查询和统计
4. **流程规范**：规范设备信息维护和校验记录管理流程

### 2.4 业务痛点

- 特种设备信息分散，难以统一管理
- 检验到期提醒不及时，存在合规风险
- 校验记录纸质存档，查询不便
- 设备状态变更缺乏追溯机制

## 4. 详细功能说明

### 4.1 设备台账列表查询

#### 4.1.1 功能概述

**需求编号**：REQ-SE-001

**功能描述**：
作为安全管理员，我可以查看特种设备台账列表，
以便了解企业内所有特种设备的基本信息和状态。

**前置条件**：
- 用户已登录系统
- 用户具有台账管理模块的访问权限

#### 4.1.2 查询条件

| 字段 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| 所属厂站 | 下拉选择 | 否 | 选择系统中已存在的厂站进行筛选 |
| 设备名称/编码 | 文本输入 | 否 | 支持设备名称或编码的模糊搜索 |
| 校验状态 | 下拉选择 | 否 | 选项：正常、预警、已过期 |
| 使用状态 | 下拉选择 | 否 | 选项：在用、停用 |

#### 4.1.3 列表展示字段

| 字段 | 说明 | 排序 | 宽度 |
|-----|------|------|------|
| 所属厂站 | 设备所属的厂站名称 | 支持 | 120px |
| 设备名称 | 特种设备的名称 | 支持 | 150px |
| 设备编码 | 设备的唯一编码 | 支持 | 120px |
| 设备类型 | 如：锅炉、压力容器、电梯等 | 支持 | 120px |
| 所属工艺段 | 设备所在的工艺段 | 支持 | 120px |
| 所属构筑物 | 设备所在的建筑物/构筑物 | 支持 | 120px |
| 型号规格 | 设备的型号和规格参数 | 支持 | 120px |
| 设备注册代码 | 监管部门的注册代码 | 支持 | 150px |
| 使用证编号 | 使用登记证书编号 | 支持 | 150px |
| 生产日期 | 设备的生产日期 | 支持 | 120px |
| 最近一次校验日期 | 上次检验的日期 | 支持 | 150px |
| 下次校验日期 | 下次应检验的日期 | 支持 | 150px |
| 校验状态 | 正常/预警/已过期（带颜色标签） | 支持 | 100px |
| 安全管理责任人 | 负责该设备安全的人员 | 支持 | 130px |
| 操作 | 编辑、详情、校验记录按钮 | - | 200px |

#### 4.1.4 校验状态说明

| 状态 | 标识颜色 | 业务规则 |
|-----|---------|---------|
| 正常 | 绿色(success) | 下次校验日期 > 当前日期 + 30天 |
| 预警 | 黄色(warning) | 下次校验日期 <= 当前日期 + 30天 且 > 当前日期 |
| 已过期 | 红色(danger) | 下次校验日期 <= 当前日期 |

#### 4.1.5 使用状态说明

| 状态 | 标识颜色 | 说明 |
|-----|---------|------|
| 在用 | 绿色(success) | 设备正在使用中 |
| 停用 | 灰色(info) | 设备已停止使用 |

#### 4.1.6 分页规则

- 默认每页显示：20条
- 可选分页：10/20/50/100条
- 显示总记录数
- 支持快速跳转页码

#### 4.1.7 交互说明

1. **搜索**：点击搜索按钮或按Enter键触发查询
2. **重置**：清空所有查询条件，恢复默认状态
3. **行点击**：点击表格行打开设备详情弹窗
4. **分页**：切换页码或每页条数时自动刷新数据

#### 4.1.8 验收标准

```gherkin
场景：查询特种设备台账
  Given 用户已进入特种设备台账页面
  When 用户输入查询条件并点击搜索
  Then 系统显示符合条件的设备列表
  And 列表按更新时间倒序排列

场景：校验状态显示
  Given 设备下次校验日期为30天后
  When 用户查看设备列表
  Then 该校验状态显示为"预警"黄色标签

场景：分页浏览
  Given 设备总数为100条
  When 用户设置每页显示20条
  Then 列表显示第1页的20条数据
  And 分页组件显示共5页
```

### 4.2 设备信息编辑

#### 4.2.1 功能概述

**需求编号**：REQ-SE-002

**功能描述**：
作为安全管理员，我可以编辑特种设备信息，
以便维护准确的设备台账数据。

**前置条件**：
- 用户已登录系统
- 用户具有设备编辑权限
- 设备已存在于系统中

#### 4.2.2 表单结构

表单分为三个区域：

**1. 基础信息区（只读）**

| 字段 | 说明 |
|-----|------|
| 所属厂站 | 从设备主数据获取，不可编辑 |
| 设备名称 | 从设备主数据获取，不可编辑 |
| 设备编码 | 从设备主数据获取，不可编辑 |
| 设备类型 | 从设备主数据获取，不可编辑 |
| 所属工艺段 | 从设备主数据获取，不可编辑 |
| 所属构筑物 | 从设备主数据获取，不可编辑 |
| 型号规格 | 从设备主数据获取，不可编辑 |
| 制造单位 | 从设备主数据获取，不可编辑 |
| 生产日期 | 从设备主数据获取，不可编辑 |

**2. 特种设备信息区（可编辑）**

| 字段 | 类型 | 必填 | 校验规则 |
|-----|------|-----|---------|
| 设备注册代码 | 文本 | 是 | 长度不超过50字符 |
| 使用证编号 | 文本 | 是 | 长度不超过50字符 |
| 吨数 | 数字 | 否 | 大于等于0，保留2位小数 |
| 设计使用年限 | 数字 | 是 | 1-100年 |
| 使用状态 | 单选 | 是 | 选项：在用、停用 |
| 使用地点 | 文本 | 是 | 长度不超过100字符 |
| 检验周期 | 数字 | 是 | 1-120月 |
| 安全管理责任人 | 文本 | 是 | 长度不超过50字符 |

**3. 校验记录区（展示+操作）**

| 字段 | 说明 |
|-----|------|
| 最近一次校验日期 | 自动计算，显示最近一次校验记录的日期 |
| 下次校验日期 | 自动计算，最近一次校验日期 + 检验周期 |
| 校验状态 | 根据下次校验日期自动计算 |
| 历史校验记录 | 列表展示，支持排序 |

#### 4.2.3 历史校验记录列表

| 字段 | 说明 | 排序 |
|-----|------|------|
| 检验日期 | 校验执行的日期 | 支持升降序 |
| 检验单位 | 执行校验的机构名称 | - |
| 检验记录 | 校验结果的详细描述 | - |
| 校验附件 | 支持下载的附件链接 | - |

#### 4.2.4 新增校验记录

**触发方式**：点击"新增校验记录"按钮

**表单字段**：

| 字段 | 类型 | 必填 | 校验规则 |
|-----|------|-----|---------|
| 检验日期 | 日期选择 | 是 | 不能晚于今天 |
| 检验单位 | 文本 | 是 | 长度不超过100字符 |
| 检验记录 | 多行文本 | 是 | 长度不超过500字符 |
| 校验附件 | 文件上传 | 否 | 支持图片、PDF、Word，单个不超过10MB |

**业务规则**：
1. 新增校验记录后，自动更新"最近一次校验日期"
2. 根据检验周期自动计算"下次校验日期"
3. 重新计算"校验状态"
4. 校验记录按检验日期倒序排列

#### 4.2.5 交互说明

1. **打开编辑**：点击列表"编辑"按钮，打开编辑弹窗
2. **表单校验**：提交时校验必填项和格式
3. **新增记录**：在编辑弹窗内可新增校验记录
4. **提交保存**：校验通过后保存数据，刷新列表
5. **取消关闭**：关闭弹窗不保存修改

#### 4.2.6 验收标准

```gherkin
场景：编辑设备信息
  Given 用户点击设备的"编辑"按钮
  When 用户修改设备注册代码并提交
  Then 系统保存修改并提示"更新成功"
  And 列表数据自动刷新

场景：新增校验记录
  Given 用户在编辑弹窗中
  When 点击"新增校验记录"并填写信息提交
  Then 系统保存校验记录
  And 自动更新设备的校验日期和状态
  And 提示"新增校验记录成功"

场景：校验规则验证
  Given 用户提交表单时未填写必填项
  When 点击"确定"按钮
  Then 系统高亮显示未填写的必填项
  And 提示"请填写必填项"
```

### 4.4 批量导入

#### 4.4.1 功能概述

**需求编号**：REQ-SE-004

**功能描述**：
作为系统管理员，我可以通过Excel批量导入特种设备数据，
以便快速初始化或更新设备台账。

**前置条件**：
- 用户已登录系统
- 用户具有数据导入权限
- 已准备好符合模板格式的Excel文件

#### 4.4.2 导入流程

**三步向导式导入**：

**步骤1：下载模板**
- 显示导入说明和注意事项
- 提供模板下载按钮

**步骤2：填写数据**
- 显示字段填写规范
- 提供下一步按钮

**步骤3：上传文件**
- 支持拖拽上传或点击选择
- 限制文件类型：.xlsx/.xls
- 限制文件大小：不超过10MB
- 显示导入结果

#### 4.4.3 导入字段规范

| 字段 | 必填 | 格式要求 |
|-----|------|---------|
| 所属厂站 | 是 | 系统中已存在的厂站名称 |
| 设备名称 | 是 | 长度不超过50字符 |
| 设备编码 | 是 | 唯一，不能重复 |
| 设备类型 | 是 | 如：锅炉、压力容器、电梯等 |
| 设备注册代码 | 是 | 格式：TSxxxxx |
| 使用证编号 | 是 | - |
| 生产日期 | 否 | 格式：YYYY-MM-DD |
| 使用状态 | 否 | 填写"在用"或"停用"，默认"在用" |

#### 4.4.4 导入结果处理

| 结果类型 | 处理方式 |
|---------|---------|
| 全部成功 | 显示成功提示，自动关闭弹窗，刷新列表 |
| 部分成功 | 显示成功和失败数量，列出失败详情（行号+原因） |
| 全部失败 | 显示失败原因，允许重新上传 |

#### 4.4.5 交互说明

1. **打开导入**：点击列表"批量导入"按钮
2. **下载模板**：步骤1点击下载按钮
3. **填写规范**：步骤2查看填写要求
4. **上传文件**：步骤3选择或拖拽文件
5. **开始导入**：点击"开始导入"按钮
6. **查看结果**：导入完成后显示结果
7. **关闭弹窗**：点击关闭按钮

#### 4.4.6 验收标准

```gherkin
场景：成功导入数据
  Given 用户上传了正确的Excel文件
  When 点击"开始导入"
  Then 系统导入所有数据
  And 提示"导入成功"
  And 列表自动刷新显示新数据

场景：部分导入失败
  Given 用户上传了包含错误数据的Excel文件
  When 点击"开始导入"
  Then 系统导入正确数据
  And 显示失败行号和原因
  And 提示"部分导入成功"

场景：文件格式错误
  Given 用户上传了非Excel文件
  When 选择文件时
  Then 系统提示"只能上传Excel文件"
```

### 4.5 批量导出

#### 4.5.1 功能概述

**需求编号**：REQ-SE-005

**功能描述**：
作为系统用户，我可以将特种设备台账导出为Excel文件，
以便离线查看或上报。

**前置条件**：
- 用户已登录系统
- 用户具有数据导出权限

#### 4.5.2 导出规则

- **导出范围**：当前查询条件下的所有数据（不受分页限制）
- **文件格式**：Excel (.xlsx)
- **文件名格式**：`特种设备台账_YYYY-MM-DD.xlsx`
- **字段范围**：列表展示的所有字段

#### 4.5.3 交互说明

1. **点击导出**：点击"批量导出"按钮
2. **确认导出**：弹出确认对话框
3. **下载文件**：确认后自动下载Excel文件
4. **取消操作**：点击取消关闭对话框

### 4.6 校验记录管理

#### 4.6.1 功能概述

**需求编号**：REQ-SE-006

**功能描述**：
作为安全管理员，我可以管理设备的校验记录，
以便追踪设备的检验历史和合规状态。

**前置条件**：
- 用户已登录系统
- 用户具有校验记录管理权限

#### 4.6.2 功能入口

- 列表操作列"校验记录"按钮
- 详情弹窗中的校验记录区域

#### 4.6.3 校验记录弹窗

**展示内容**：
1. 设备基础信息（只读展示）
2. 特种设备信息（只读展示）
3. 当前校验状态摘要
4. 历史校验记录列表
5. 新增校验记录按钮

**记录列表字段**：

| 字段 | 说明 | 排序 |

| 校验附件 | 可下载的附件 | - |

**表单字段**：

| 字段 | 类型 | 必填 | 说明 |
