# **第三方人员管理**

所属第三方单位：下拉选择，从已创建的第三方单位中选择，单选，必填；

人员名称：弹出列表选项，根据已选所属第三方单位获取【人员证书管理】中配置的人员信息进行选择

身份证号：根据已选人员名称自动带出

工种：根据已选人员名称自动带出

# **运营项目作业管理**

作业内容：

高风险作业类型：下拉选择，选项为动火作业/有限空间作业/高处作业/吊装作业/临时用电作业/动土作业/断路作业/盲板抽堵作业，单选，必填；

作业单位：内部/外部；如果选择内部，默认填充当前用户所在的组织节点名称；如果选择外部，则从已配置的第三方单位中选择；
