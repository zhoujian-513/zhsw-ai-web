# 【产品模块】有限空间台账管理

## 一、业务概述

业务场景：对有限空间台账信息进行管理维护
适用角色：安全管理人员
业务流程：录入有限空间台账→审核有限空间台→开有限空间空间作业票可选择已有的有限空间台账
权限规则：不同角色可见按钮、可编辑字段不同

## 二、页面清单&交互链路

1. 列表页 → 点击详情 → 记录详情页
2. 列表页 → 点击新增 → 新增表单页
3. 列表页 → 点击编辑 → 修改表单页
4. 列表页 → 点击审核 → 审核表单页
5. 详情页 → 点击删除 → 删除已有的记录
6. 列表页 → 点击批量导入→下载模板→上传批量导入文件
7. 列表页 → 点击批量审核→勾选多条待审核记录→进行审核

## 三、页面1：数据列表页

### 页面布局

顶部搜索栏  + 功能按钮 + 数据表格 + 分页

### 搜索筛选项

所属厂站、有限空间编号、所在部位（构筑物）、状态（待审核、审核通过）、主要风险分析（可多选）、填报月份

### 表格展示字段

所属厂站、有限空间编号、所在部位、具体位置、状态、主要风险分析、风险管控措施、责任人、填报人、填报时间

### 页面按钮

新增、编辑、详情、删除、审核、批量导出、批量导入、批量审核

### 交互规则
- 分页默认20条/页
- 点击单据行跳转详情
- 无权限按钮自动灰显隐藏
