# 【产品模块】特种设备台账管理
## 一、业务概述
业务场景：对特种设备的基本信息、校验记录进行管理维护
适用角色：特种设备管理人员
业务流程：在设备中心录入设备 → 获取特种设备台账，补充基本信息 → 填写校验记录
权限规则：不同角色可见按钮、可编辑字段不同

## 二、页面清单&交互链路
1. 列表页 → 点击详情跳转详情页
2. 列表页 → 编辑 → 修改表单
3. 详情页 → 校验记录 → 查看已有校验记录 →新增校验记录
4. 列表页 → 点击批量导入→下载模板→上传批量导入文件

## 三、页面1：数据列表页
### 页面布局
顶部搜索栏 + 状态筛选 + 功能按钮 + 数据表格 + 分页
### 搜索筛选项
所属厂站、设备名称/编码、校验状态、使用状态
### 表格展示字段
所属厂站、设备名称、设备编码、设备类型、所属工艺段、所属构筑物、型号规格、设备注册代码、使用证编号、生产日期、最近一次校验日期、下次校验日期、校验状态、安全管理责任人
### 页面按钮
编辑、详情、校验记录、批量导出、批量导入
### 交互规则
- 状态不同颜色高亮：正常/预警/已过期
- 分页默认20条/页
- 点击单据行跳转详情
- 无权限按钮自动灰显隐藏

## 四、页面2：编辑表单页
### 表单分区
基础信息区、特种设备信息区、校验记录区
### 基础信息区
所属厂站、设备名称、设备编码、设备类型、所属工艺段、所属构筑物、型号规格、制造单位
以上信息，从设备中心自动获取，不可编辑
### 特种设备信息区
设备注册代码、使用证编号、吨数、设计使用年限（年）、使用状态（在用、停用）、使用地点、检验周期、安全管理责任人
以上信息，均可手动修改编辑
### 校验记录区
最近一次校验日期
下次校验日期（最近一次校验日期+校验周期）
校验状态（下次校验日期减去当前日期大于7天，为正常；下次校验日期减去当前日期小于等于7天。为预警；下次校验日期已超过当前日期，为已过期）
####校验记录列表
检验日期、检验单位、检验记录、校验附件



## 五、页面3：单据详情页
- 只读展示全部信息
- 历史修改记录、日志溯源

## 六、页面4：新增校验记录页
-基础信息区、特种设备信息区、校验记录区
### 基础信息区
所属厂站、设备名称、设备编码、设备类型、所属工艺段、所属构筑物、型号规格、制造单位
以上信息，从设备中心自动获取，不可编辑
### 特种设备信息区
设备注册代码、使用证编号、吨数、设计使用年限（年）、使用状态（在用、停用）、使用地点、检验周期、安全管理责任人
以上信息，不可修改编辑
### 校验记录区
最近一次校验日期
下次校验日期（最近一次校验日期+校验周期）
校验状态（下次校验日期减去当前日期大于7天，为正常；下次校验日期减去当前日期小于等于7天，为预警；下次校验日期已超过当前日期，为已过期）
####校验记录列表
检验日期、检验单位、检验记录、校验附件
点击“新增校验记录”，可新增一行校验记录信息
提交后自动更新最近一次校验日期、下次校验日期及校验状态

## 七、校验记录状态机
正常：下次校验日期减去当前日期大于7天
预警：下次校验日期减去当前日期小于等于7天
已过期：下次校验日期已超过当前日期

## 七、权限控制
1. 特种设备管理员：全部编辑、新增校验记录权限
2. 普通业务：仅查看详情
